Engineering Tools

Four specialist tools spanning UAV aerodynamics, ground control, propulsion systems, and digital twin technology — each built from first principles by a practising aerospace engineer.

01 / 04
Travan Aero Sim
Aerodynamic Simulation

A comprehensive UAV aerodynamic simulation platform built on MachUp aerodynamic data with validated lift, drag, and moment calculations. Provides multi-tab analysis across airframe geometry, propulsion integration, VTOL transition modelling, live real-time simulation, and automated report generation. Designed for fixed-wing UAV design, optimisation, and performance validation workflows.

MachUp-validated aerodynamic data engine
VTOL transition simulation module
Live simulation with real-time input editing
PDF & JSON report export
Propulsion and airframe co-analysis
Format: .zip | Platform: Win 10/11
Download
02 / 04
Travan GCS
Ground Control Station

A full-featured desktop Ground Control Station for real UAV operations and simulation. Communicates via raw MAVLink v1 and v2 protocols over serial and UDP, supporting ArduPilot, PX4, iNav, and generic MAVLink flight controllers. Features an interactive Leaflet.js mission planner, live AHRS attitude display, parameter manager, telemetry analytics dashboard, and video feed panel — all in a unified Electron-based desktop environment.

Raw MAVLink v1/v2 protocol support
ArduPilot, PX4, iNav compatibility
Interactive Leaflet.js mission planner
Live AHRS & telemetry analytics dashboard
Serial / UDP / TCP communication modes
Format: .zip | Platform: Win 10/11
Download
03 / 04
Propulsion Twin
Propulsion System Simulation

A high-fidelity propulsion system digital twin developed as a final-year project deliverable and extended into a standalone tool. Models hybrid and conventional turbofan/turboprop architectures with thermodynamic cycle analysis, component performance maps, and transient response simulation. Intended for propulsion researchers, aerospace engineering students, and systems engineers validating engine performance against design targets.

Thermodynamic cycle analysis (Brayton cycle)
Component performance map integration
Hybrid & conventional propulsion architectures
Transient response and off-design simulation
Parametric design space exploration
Format: .zip | Platform: Win 10/11
Download
04 / 04
Travan Digital Twin
Systems Digital Twin

A modular engineering digital twin framework for real-time system mirroring, telemetry fusion, and predictive analysis of UAV and aerospace platforms. Integrates sensor data streams, physics-based state estimation, and live 3D visualisation to create a dynamic virtual replica of a physical system. Suitable for fleet health monitoring, anomaly detection, mission replay, and engineering research into cyber-physical system architectures.

Real-time sensor telemetry fusion
Physics-based state estimation engine
Live 3D system visualisation
Anomaly detection and health monitoring
Mission replay and post-flight analysis
Format: .zip | Platform: Win 10/11
Download

Download Instructions: Each tool is packaged as a compressed .zip archive located in the assets/ folder. Extract the archive to a directory of your choice and run the .exe file directly — no additional installation required. If Windows Defender flags the executable, click "More Info → Run Anyway". Tools are unsigned executables and pose no risk.

About the Author

Travan Labs is the engineering software division of the Travan brand — a portfolio of production-grade tools built by Dickson Tawiah Aman, an Aerospace Engineer and Associate Member of the Royal Aeronautical Society (ARAeS).
